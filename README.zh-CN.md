## 安装使用

- 获取项目代码

```bash
git clone https://github.com/3052971491/foundation.git
```

- 下载依赖

```bash
cd foundation

yarn install

```

- 打包

```bash
yarn build
```

- 运行

```bash
yarn dev
```
