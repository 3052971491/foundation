## Features

- High quality Vue framework out of the box。
- Use VUE3, Vite, TS and other mainstream technologies。

## Install and use

- Get the project code

```bash
git clone https://github.com/3052971491/foundation.git
```

- Installation dependencies

```bash
cd foundation

yarn install

```

- build

```bash
yarn build
```

- run

```bash
yarn dev
```
